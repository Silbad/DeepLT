## Steps for dev
* Install Node.js & Yarn [https://yarnpkg.com/en/docs/install](https://yarnpkg.com/en/docs/install)
* Launch a cmd window
* Open your clone folder :
```
cd C:\path_to_folder
```
* Get dependencies :
```
yarn
```
* Launch Gulp tasks :
```
yarn gulp
```
* Launch the WebExtension :
```
yarn web-ext run
```
